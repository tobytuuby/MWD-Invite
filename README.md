# MWD Invite to Play

A World of Warcraft addon that helps you quickly invite guild members to your party or raid.

## Features
- Searchable guild roster with selection
- Invite Party, Invite Raid, and Invite (auto-convert to raid for 5+)
- Minimap button toggle
- Guild & Communities top-bar button to open the window
- Optional “Main” column from Guild Roster Manager (GRM)

## Installation
1. Download or clone this repository.
2. Copy the `MWDInvite` folder into your WoW `Interface\AddOns` directory.
3. Enable the addon from the character select AddOns menu.

## Usage
- `/mwdinv ui` toggles the invite window
- Use the search box to filter, then click rows to select members
- Use Invite Party/Raid/Invite buttons to send invites

## Development
- No build step required. Edit the Lua/XML files directly.
