# MWDInvite

A World of Warcraft addon that improves calendar and invite management.

## Features
- Streamlined invite workflows
- Clear status indicators for events and invites
- Lightweight and easy to use

## Installation
1. Download or clone this repository.
2. Copy the `MWDInvite` folder into your WoW `Interface\AddOns` directory.
3. Enable the addon from the character select AddOns menu.

## Usage
- Open the in-game calendar and use the addon controls to manage invites.

## Development
- No build step required. Edit the Lua/XML files directly.
